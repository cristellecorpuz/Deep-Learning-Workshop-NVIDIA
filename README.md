# Deep-Learning-Workshop-NVIDIA
This repository contains my assessment from the Deep Learning Fundamentals course by NVIDIA.
---
📌 Overview

The course covers core deep learning concepts such as:

1. Neural networks and CNNs
2. Image classification
3. Data augmentation
4. Transfer learning and fine-tuning
5. Model evaluation and optimization

All datasets and images used in this repository were provided by NVIDIA as part of the course materials.

---
🧪 Final Assessment
In the final project, I trained a deep learning model to classify fresh vs. rotten fruit by applying transfer learning and data augmentation to achieve atleast 92% accuracy.

---
✅ Results
- Validation Loss: 2.0618
- **Validation Accuracy: 94.53%**
